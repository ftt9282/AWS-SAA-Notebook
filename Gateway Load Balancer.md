The Gateway Load Balancer is used to route traffic to what is referred to as 3rd party network virtual appliances.

These virtual applications are things like firewalls, intrusion detection and prevention systems, packet inspection, etc.

It operates at Layer 3 (Network Layer) and uses the **GENEVE protocol on port 6081** (this might be on the exam)

Can have the following Target Groups:
- EC2 Instances
- IP Addresses
