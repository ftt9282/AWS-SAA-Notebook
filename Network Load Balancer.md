The Network Load Balancer uses TCP/UDP. In the exam when you see "TCP" or "UDP" think Network Load Balancer.

Additionally, Network Load Balancers provide pretty extreme performance, and they can only have one static IP per AZ. So on the exam if you see anything related to high performance or "one AZ" think Network Load Balancer.

Network Load Balancer can have 3 target groups:
- EC2 Instances
- Private IP Addresses
- [[Application Load Balancer]]

 Health Checks for the Network Load Balancer support **TCP, HTTP, and HTTPS**.
