Route 53 is a scalable, fully managed *authoritative* DNS
(Authoritative means that the customer can update the DNS records). Route 53 is also a domain registrar, so you can register something like `example.com` with Route 53.

Some Route 53 Record types that you'll need to know for the exam: [[Route 53 Record Types]]

Route 53 also has various Routing Policies that it uses to respond to DNS queries. Those policies are:
- [[Simple Routing]]
- [[Weighted Routing]]
- [[Failover Routing]]
- [[Latency Based Routing]]
- [[Geolocation Routing]]
- [[Multi-Value Answer Routing]]
- [[Geoproximity Routing]]
- [[IP-Based Routing]]
